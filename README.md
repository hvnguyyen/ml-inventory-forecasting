# ml-inventory-forecasting

DAT255 project on uncertainty-aware inventory forecasting.

## Project idea
We use deep learning to forecast retail demand and test whether uncertainty-aware forecasts can improve inventory restocking decisions.

## Dataset
- M5 Forecasting dataset

## Goal
- Train at least one deep learning model from scratch
- Compare point forecasts and uncertainty-aware forecasts
- Evaluate both forecast accuracy and inventory performance
